# IRCTC Re-Design 🚄

A modernized, premium, and fully responsive redesign of the Indian Railway Catering and Tourism Corporation (IRCTC) web portal. This project reimagines the booking experience with a focus on aesthetics, user-centric design, and modern web capabilities.

**🚀 Live Deployment:** [https://irctc-re-design.vercel.app/](https://irctc-re-design.vercel.app/)

## ✨ Key Features

- **Premium UI/UX:** A sleek interface featuring dynamic glassmorphism, smooth micro-animations, and curated color palettes designed for visual excellence.
- **Theme-Aware Hero Section:** The homepage automatically adapts its stunning hero background based on the user's active theme (Light/Dark mode).
- **AskDisha 2.0 AI Assistant:** A highly interactive, floating support chatbot that provides contextual help, smart quick replies, and step-by-step guidance for booking, PNR checks, and more.
- **Full Hindi Localization (हिन्दी):** Comprehensive bilingual support. Users can seamlessly toggle between English and Hindi, instantly translating everything from the homepage search form and support FAQs to the intelligent AskDisha chatbot.
- **Comprehensive Service Pages:** Deeply integrated dummy-data views for Train Search results, PNR Status tracking, E-Catering (Meals), IRCTC E-Wallet, Help & Support, and My Trips.
- **Seamless Navigation:** A robust, sticky navigation bar providing instant access to premium services, alerts, and language preferences.

## 🛠️ Tech Stack

- **Frontend Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v3 (Custom theme configurations, complex gradients, dynamic responsive classes)
- **State Management:** React Context API (e.g., `LanguageContext` for centralized global translations)
- **Routing:** React Router v6
- **Icons & Typography:** Google Material Symbols, Modern UI Fonts (Inter/Outfit)
- **Deployment:** Vercel (Configured via `vercel.json` for React/Vite SPA routing)

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/riturajjhaba938/IRCTC-Re-Design.git
   ```

2. **Navigate to the frontend directory**
   ```bash
   cd IRCTC-Re-Design/frontend
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate your browser to `http://localhost:5173`.
